Splash Screen

A splash screen page for a website focusing on Sustainable Development Goal #4.

Header and Navigation Bar:The header contains a navigation bar with links to different sections of the website such as Home, Overview, Shop, Volunteer, and About Us. The navigation bar has a red background color and changes color on hover.

Splash Container:This section contains the main content of the splash screen. It's designed to occupy the full height of the viewport (100vh) and is centered vertically and horizontally using flexbox. It displays the Sustainable Development Goal #4 title along with a brief description emphasizing "Quality Education."

Loader Animation:Below the description, there's a spinning loader animation implemented using CSS keyframes. This indicates to the user that some process is ongoing, possibly loading resources or redirecting.

Group Members:Lists the group members involved in the project along with their names and student IDs.

Script:A script tag with JavaScript code is included at the end of the body. This script redirects the user to the "home.html" page after a delay of 4000 milliseconds (4 seconds). This is a common technique used for splash screens to briefly display branding or information before directing the user to the main content.

The Shop

User-friendly interface for browsing and purchasing products related to quality education, with features for customization and easy checkout.

Main Section:Displays a collection of products related to the theme.Each product has an image, name, price, customization options (size and color), and an "Add to Cart" button. Products are arranged in a grid layout.

The Merchandise Range:I personally created every product sold in the shop, from designing and editing each item to uploading them onto my Flickr account.

Order Summary Section: Displays the items added to the cart along with their details (name, price, quantity, size, and color).Shows the total price of the items in the cart. Includes an "Undo" button to remove the last item added to the cart.

Checkout Section:Contains a form for the user to fill out their name, email, address, and phone number for checkout.Upon submission, it displays the order summary and user details for confirmation.

JavaScript:Handles the functionality of adding items to the cart, updating the order summary, and displaying user feedback upon checkout. Utilizes event listeners to respond to user interactions such as clicking "Add to Cart" buttons and submitting the checkout form.
